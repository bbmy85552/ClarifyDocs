## 1. 註冊 Telegram 開發者帳號 
- 登錄 Telegram 帳號，在 Telegram 上搜索並關注 `@BotFather` 機器人。 
- 使用 `/newbot` 命令建立一個新的機器人，輸入機器人的名稱和用戶名。 
- BotFather 會返回一個 API Token，這個 Token 是後續調用 Telegram API 的憑證。 

## 2. 獲取 API Token 
- 在建立機器人後，BotFather 會傳送給你一個 API Token，格式如 `123456789:ABCDEF1234567890abcdef1234567890`。 
- 保存這個 Token，因為它是調用 Telegram API 的憑證。 

## 3. 在connct頁面與Chatbot進行綁定
- 上傳 bot token 與 chatbot 進行綁定。 
- 綁定成功後可以在 telegram 上面搜索機器人名稱，並開始進行聊天。
